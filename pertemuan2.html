Langkah-langkah Hosting di Layanan Hosting Gratis

  1. Pilih layanan hosting gratis yang sesuai dengan kebutuhan Anda. Beberapa contoh layanan hosting gratis populer adalah 000webhost, InfinityFree, dan GitHub Pages.
  2. Buka situs web penyedia layanan hosting gratis dan buat akun pengguna baru.
  3. Setelah mendaftar, masuk ke akun Anda dan cari opsi untuk membuat atau tambahkan website baru.
  4. Pilih opsi untuk menggunakan nama domain gratis yang disediakan oleh layanan hosting atau gunakan nama domain yang Anda miliki.
  5. Jika menggunakan nama domain gratis, pilih nama domain yang tersedia dari opsi yang diberikan. Jika menggunakan nama domain sendiri, ikuti petunjuk untuk mengatur DNS (Domain Name System) agar mengarah ke layanan hosting gratis tersebut.
  6. Upload file website Anda ke layanan hosting. Biasanya, Anda dapat menggunakan fitur seperti FTP (File Transfer Protocol) atau File Manager yang disediakan oleh layanan hosting.
  7. Pastikan file website Anda ditempatkan di direktori root atau direktori yang ditentukan untuk file HTML utama.
  8. Setelah file website diunggah, tunggu beberapa saat agar perubahan dapat tersimpan dan diterapkan dengan baik.
  9. Periksa website Anda dengan membuka nama domain atau alamat yang diberikan oleh layanan hosting. Pastikan website Anda dapat diakses dengan baik.
  10. Jika diperlukan, sesuaikan pengaturan atau konfigurasi tambahan yang disediakan oleh layanan hosting gratis untuk mengoptimalkan website Anda.
